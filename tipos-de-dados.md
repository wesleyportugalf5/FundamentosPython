Tipos de dados 

O Python é uma linguagem de tipagem dinâmica, ou seja, não precisamos definir o tipo de uma variável no momento da sua criação, esse tipo dependerá do conteúdo que essa variável armazenará. Por exemplo, se criamos uma variável e inserimos nela um valor numérico, logo, essa variável será do tipo numérico. 

No Python, temos alguns tipos nativos, são eles: 

Int: Números inteiros 
Str: Cadeia de caracteres 
Bool: Dados booleanos (True ou False) 
List: Estrutura de dados para armazenamento de conjunto de elementos 
Dict: Dicionários 
Files: Arquivos
